# Typometer⌨️

This is a simple web-app built using React JS which tests your typing speed.
You can use it [here](https://anurag-pratik.github.io/typometer/) .

## Installation📥

Use [npm](https://www.npmjs.com/) to install the dependencies.

```bash
npm install
```

## Usage🖥️
